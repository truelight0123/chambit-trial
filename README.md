## ✅ 구현된 기능

### 메인 페이지 (index.html)
- **고정 헤더 네비게이션** — 스크롤 시 배경/그림자 자동 적용, 모바일 햄버거 메뉴
- **히어로 섹션** — 가격 정보 박스, CTA 버튼 2개, 하단 통계 바
- **왜 참빛인가** — 4개 카드 그리드 (아이콘 + 설명)
- **추천 대상** — 5가지 아이콘 카드
- **세대별 활용** — 4개 카드 (성장기 / 젊은세대 / 4050 / 부모님), 키워드 배지
- **핵심 특징** — 5개 아이콘 배지
- **가격·구독 안내** — 소비자가 / 등록비 / 월 구독료 / 유의사항
- **자료·연구 흐름** — 4개 리서치 카드 (상단 컬러바)
- **무료 상담 신청 폼** — 유효성 검사, 로딩 상태, 성공 화면, Table API 저장
- **추천코드** — URL 파라미터(`?ref=코드`) 수신 → localStorage 저장 → 폼 자동 입력
- **스크롤 애니메이션** — IntersectionObserver 기반 fade-in
- **상단 이동 버튼** — 스크롤 400px 이상 시 노출
- **반응형 디자인** — 모바일 / 태블릿 / PC 완전 대응

### 관리자 페이지 (admin.html)
- **요약 카드** — 전체 / 신규 / 확인중 / 완료 건수
- **데이터 테이블** — 신청 목록 페이지네이션 (15건/페이지)
- **검색 및 필터** — 이름·연락처·지역 검색, 상태·유형 필터
- **상세 모달** — 신청 정보 전체 보기
- **상태 변경** — 신규 / 확인중 / 상담완료 / 설치완료 PATCH
- **관리자 메모** — 건별 메모 PATCH 저장
- **삭제** — 개별 레코드 삭제
- **자동 새로고침** — 30초 간격

## 🔗 주요 URI

| 경로 | 설명 |
|------|------|
| `/index.html` | 메인 랜딩 페이지 |
| `/index.html?ref=추천코드` | 추천코드 포함 랜딩 |
| `/admin.html` | 관리자 상담 신청 관리 페이지 |

## 🗄️ 데이터 모델

### `consultations` 테이블

| 필드명 | 타입 | 설명 |
|--------|------|------|
| `id` | text | 자동 생성 UUID |
| `name` | text | 신청자 이름 |
| `phone` | text | 연락처 |
| `region` | text | 지역 |
| `consult_type` | text | 상담 유형 |
| `install_target` | text | 설치 대상 |
| `purpose` | text | 사용 목적 |
| `message` | rich_text | 문의 내용 |
| `referral_code` | text | 추천코드 |
| `status` | text | 처리 상태 (신규/확인중/상담완료/설치완료) |
| `admin_memo` | rich_text | 관리자 메모 |
| `created_at` | (자동) | 신청 일시 (ms) |
